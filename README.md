# Football Analysis

## Overview
This repository contains football analysis exercise. It performs an Exploratory Data Analysis (EDA) on over 150 years of international football matches using Python, Pandas, and Matplotlib.

## Dataset
Source: [Kaggle: International Football Results (1872–2024)](https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017)

##  Project Structure
The analysis is broken down into the following key sections:

- Basic Exploration: Calculating total matches, identifying date ranges, and finding unique countries/teams.

- Goals Analysis: Analyzing average goals, identifying the highest-scoring matches, and comparing home vs. away scoring trends.

- Match Results: Categorizing match outcomes (Home Win, Away Win, Draw) to calculate win percentages and historically successful teams.

- Visualization: Generating histograms and bar charts to visually communicate the findings.
